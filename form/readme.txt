1. 設計一個網頁，內含一個表單(form)，屬性設定如下：
<form method="post" action="http://ycchen.im.ncnu.edu.tw/join.php">
...
</form>

2. 表單需含有以下表單元素(屬性):
  (1). text
  (2). password
  (3). checkbox
  (4). radio
  (5). select/option
  (6). submit
  (7). reset

3. 於必須填入資料之表單元素，加上required屬性。
4. 使用placeholder屬性，加入提示文字。
5. 在checkbox, radio加入checked屬性，設定預設選項。
6. 在option加入selected屬性，設定預設選項。
7. 在checkbox, radio使用label元素，可以點選文字。

8. 使用css進行對齊設定
  (1). 提示文字放在span元素中
  (2). 將span之display屬性設為inline-block
  (3). 設定span之寬度(width)及文字對齊(text-align)屬性
